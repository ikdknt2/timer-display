# Timer Display

G5風のタイマー表示を再現したシンプルなWebツールです。  
URLのパラメータを使ってタイムを表示できます。

---

## 🔗 デモ

https://ikdknt2.github.io/timer-display/?id=123456

---

## 🎯 使い方

URLの `?id=` に6桁の数字を入れるだけで表示されます。

### 例


?id=123456 → 1:23.456

?id=059847 → 59.847

?id=104023 → 1:04.023


---

## ⚙️ 仕様

- 6桁固定（mmssSSS）
- 分・秒・ミリ秒を分解して表示
- 0の桁はスペースに置き換え（見た目調整）
- 桁ごとに分離しているためズレなし
- CSSで位置固定
